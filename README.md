# 📌 Aplicação de Tarefas

Uma aplicação React desenvolvida para gerenciamento de tarefas (To-Do List), permitindo adicionar, visualizar, editar e excluir tarefas de forma prática e organizada.

Este projeto faz parte de um desafio prático do curso **DevStart**, com foco no aprendizado de conceitos fundamentais e intermediários do React, incluindo rotas, Context API, hooks e persistência de dados.

---

## 🚀 Tecnologias Utilizadas

* ⚛️ React
* 🟨 JavaScript
* 🎨 CSS
* 📦 React Router DOM
* 🧠 Context API
* 💾 LocalStorage

---

## 📷 Funcionalidades

✅ Adicionar tarefas

✅ Visualizar lista de tarefas

✅ Editar tarefas

✅ Excluir tarefas

✅ Navegação entre páginas com React Router

✅ Gerenciamento global de estado com Context API

✅ Persistência de dados utilizando LocalStorage

## 🎯 Objetivo do Projeto

O objetivo deste projeto foi praticar conceitos importantes do React através da criação de uma aplicação completa de gerenciamento de tarefas, reforçando habilidades em roteamento, manipulação de estado, persistência de dados e organização de componentes.

Além da funcionalidade proposta no desafio, foi adicionada uma estilização personalizada utilizando CSS para melhorar a experiência visual da aplicação.
